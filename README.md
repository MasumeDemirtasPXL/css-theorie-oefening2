## Concept
In deze oefening leer je de basis van CSS syntax en hoe je een extern CSS-bestand koppelt aan je HTML. Je oefent met het aanpassen van stijlen via element-selectors en class-selectors, en leert het belang van gedocumenteerde code door middel van commentaren.

## Commitrichtlijnen
Gebruik duidelijke commitboodschappen om je voortgang bij te houden:
* `CSS-2.0-01: HTML basisstructuur opgezet`
* `CSS-2.0-02: Extern CSS-bestand gekoppeld`
* `CSS-2.0-03: Styling voor h1 en paragrafen toegepast`
* `CSS-2.0-04: Highlight class gestyled en commentaren toegevoegd`

---

## Algemene instructies
* Werk in de bestanden `index.html` en `style.css`.
* Zorg dat de bestanden in dezelfde map staan.
* Koppel het CSS-bestand in de `<head>` van je HTML.
* **Geen inline CSS:** Gebruik uitsluitend het externe `style.css` bestand.

---

## Opdracht: Aanpassingen in style.css

Voer de volgende taken uit om de styling te voltooien:

| Onderdeel | Actie |
| :--- | :--- |
| **Kleur** | Pas de tekstkleur van het `<h1>`-element aan naar **rood**. |
| **Tekststijl** | Zorg dat alle `<p>`-elementen **cursief** (`italic`) worden weergegeven. |
| **Rand** | Voeg een **rand** (border) toe aan de class `.highlight`. |
| **Documentatie** | Voeg boven **elke** CSS-selector een commentaar-regel toe die uitlegt wat de regel doet. |

---

## Checklist voor indiening
- [ ] HTML en CSS staan in dezelfde map.
- [ ] CSS-bestand is correct gekoppeld via de `<link>` tag.
- [ ] De `<h1>` is rood.
- [ ] Alle paragrafen zijn cursief (`font-style: italic`).
- [ ] De `.highlight` paragraaf heeft een zichtbare rand.
- [ ] Elke CSS-selector is voorzien van een verklarend commentaar.
- [ ] Er zijn geen `style=""` attributen gebruikt in de HTML-code.
